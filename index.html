Install Hyperion

How to install ?

In the first step, since Hyperion relies on the Radiance simulation engine, you need to install Radiance. To install it, click on the link provided in the download box to be directed to the Radiance GitHub page. Then, go to the Assets section of the latest released version, download the .exe file, and install it.

In the second step, download the Hyperion plugin from the download box and place the component folder in the _ Grasshopper directory.

If you are not familiar with how to model for Radiance, we recommend using the Ladybug Tools (Pollination) plugin. The download link for it is also provided below.
